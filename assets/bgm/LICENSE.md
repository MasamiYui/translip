# Bundled Commentary BGM Placeholders

This directory ships **algorithmically synthesised** placeholder background
music tracks for the commentary recap pipeline. They are intentionally low-fi
loops generated by [`scripts/build_bgm_placeholders.py`](../../scripts/build_bgm_placeholders.py)
from pure-Python `math.sin` / envelope code — they contain **no samples,
loops, stems, or anything else from a third-party recording**.

`python scripts/build_bgm_placeholders.py` produces byte-identical output.

The placeholders are designed only to validate the **plumbing** (renderer
filtergraph, sidechain ducking, preview endpoint, frontend picker). For any
production / publishing work you should drop replacement tracks at the same
path. The filename must match the preset's `asset_filename` recorded in
[`src/translip/commentary/bgm.py`](../../src/translip/commentary/bgm.py):

  recap video must obtain the appropriate licenses from the rights holder /
  publishing organisation in your jurisdiction.

## Audio specs

The placeholders are generated as:

- WAV / mono / 16-bit PCM / 44.1 kHz
- 30 seconds each (the renderer loops them via `ffmpeg -stream_loop -1`)
- Peak amplitude pre-attenuated to ~-15 dBFS so the renderer's
  `bgm_gain_db` knob has plenty of dynamic range to play with
