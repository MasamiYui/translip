# AI 助手

说不清该用哪些步骤？把目标用一句话告诉 **AI 助手**。它会自动把多个原子工具串成一条调用链，**先把计划摆给你看**，确认后再逐步执行，最后汇总产物给你下载。

入口：界面右下角的悬浮图标随时唤起助手；历史记录在「任务中心 → **AI 任务**」。

![AI 助手：一句话编排原子能力](/guide/flow-assistant.svg)

*图 1：先规划、再确认、后执行——计划全程可见，参数可改，绝不黑箱直接跑。*

## 怎么用

![AI 助手对话窗口](/guide/ai-assistant-widget.png)

*图 2：助手窗口——顶部是工作方式说明，中间是示例指令，底部是输入框（可附文件）。*

1. **描述目标**——在输入框里说要做什么，例如「识别这个视频的日语字幕并转成中文字幕」「把这段视频配成英文配音」。可点回形针**附上文件**。
2. **看计划**——助手用 DeepSeek 把目标拆成一条**调用链**（如：字幕识别 → 翻译 → 合成 → 配音渲染），以流程图展示每一步。信息不足时它会**反问澄清**而不是乱猜。
3. **确认 / 改参数**——计划里每一步的参数都可编辑；看清楚后点「**确认执行**」。
4. **逐步执行**——实时显示每一步的状态与进度，可随时「停止执行」。
5. **下载产物**——跑完后列出所有最终文件供下载。

## 示例指令

助手窗口里内置了一批示例，按场景给你灵感：

| 场景 | 例子 |
| --- | --- |
| 字幕 | 识别这个视频的日语字幕并转成中文字幕 |
| 配音 | 把这个视频翻译成中文配音 |
| 音频 | 分离人声和背景音 / 识别这个文件的语言 |
| 视频 | 给这个视频压上水印 / 探测这个文件的媒体信息 |

## AI 任务历史

每次对话都会留档在「**AI 任务**」里。

![AI 任务历史列表](/guide/ai-tasks.png)

*图 3：AI 任务历史——可搜索、按状态筛选，查看每次的调用链、产物，并可重跑 / 取消。*

点一条进详情，能看到原始请求、还原出的调用链、每步产物下载链接，以及失败时的错误信息；可重跑同一条计划或删除记录。

> ⚠️ AI 助手需要 **DeepSeek API Key**（在[全局设置 → API 密钥](/guide/settings)里配置）。未配置时会提示并给出「去设置」入口。
>
> 助手只**编排已有的原子工具**，不会凭空生成新能力——它能做的，都是[原子工具](/guide/atomic-tools)里列出的那些。
