# 原子工具

不想跑整条流水线、只想要某一步的产物？用「**原子工具**」。每个工具都是一个独立的单步作业：上传输入 → 设置参数 → 运行 → 下载产物。它和流水线共用同一批后端，只是少了编排——你自己决定跑哪一步、怎么串。

入口：左侧「**原子工具集**」。工具按音频 / 语音 / 视频 / 字幕分类列出。

![原子工具集：按类别列出的单步工具](/guide/atomic-tools.png)

*图 1：原子工具集，按处理领域分组。*

## 用一个工具

点开任意工具，流程都一样：

![单个原子工具：人声 / 背景分离](/guide/tool-separation.png)

*图 2：以「人声 / 背景分离」为例——左侧上传与参数，右侧结果。*

1. **上传输入**——拖拽或点选文件，显示文件名与大小。
2. **设置参数**——每个工具的参数不同（语言、后端、质量、模式…），不确定就用默认。按钮旁会提示缺什么（如某 TTS 后端「需要上传参考音频」）。
3. **开始处理**——点运行；进度条显示当前阶段。
4. **拿结果**——产物可内嵌预览并下载；有些工具还提供**跨工具动作**，例如把分离出的人声「用于语音合成」，一键带去下一个工具。

部分工具带**实时预览面板**：视频裁剪可在时间轴上拖拽设定起止点，水印可所见即所得地调位置 / 透明度，字幕识别可叠加显示检测框。

![视频裁剪工具：时间轴拖拽](/guide/tool-video-trim.png)

*图 3：视频裁剪——拖动手柄设定起止点，支持精确剪切与快速无损两种模式。*

## 任务历史

每次运行都会记录在「**任务中心 → 原子任务**」里。

![原子任务历史列表](/guide/atomic-jobs.png)

*图 4：原子任务历史——可搜索、按状态 / 工具筛选，逐条重跑 / 停止 / 删除。*

- 按工具、状态、文件名筛选与搜索，支持分页与批量删除。
- 点一条进详情：状态、进度、耗时、输入文件，以及按主产物 / 预览 / 诊断分组的产物面板（可内嵌播放、展开预览、下载）。
- 从详情可「再次打开工具」「重跑」「停止」或删除记录。

> 💡 原子工具适合**手动串联**：A 工具的产物下载后喂给 B 工具。如果嫌手动麻烦，让 [AI 助手](/guide/ai-assistant)用一句话替你把链路串起来。
