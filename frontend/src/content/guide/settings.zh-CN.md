# 全局设置与模型管理

「全局设置」是配置中心：在这里填 API 密钥、下载缺失的模型、查看系统信息、调整流水线的高级默认值。入口在左侧「**全局设置**」。

页面分两个大标签：**通用**（日常需要的密钥、模型、系统信息）和**高级配置**（各阶段的默认参数，普通用户一般用不到）。

![全局设置页面](/guide/settings.png)

*图 1：全局设置——左侧分区导航，右侧是当前分区的内容。*

## 通用设置

| 分区 | 作用 |
| --- | --- |
| **系统信息** | 查看 Python 版本、平台、计算设备、缓存目录与缓存大小；可一键清理缓存。 |
| **API 密钥** | 配置 HuggingFace 令牌（门控模型）、DeepSeek（翻译 / 助手 / 评测）、TMDb（作品导入）。 |
| **模型管理** | 查看各模型状态并下载缺失项。 |
| **更新日志** | 浏览各版本的更新说明。 |
| **关于** | 项目简介。 |

每个密钥分区都能**保存**并**测试连接**，旁边有「已配置 / 未配置」状态。

> ⚠️ 保存 HuggingFace 令牌后通常需要**重启服务**才生效，界面会提示。

## 下载模型

translip 默认在本地跑模型。首次使用某能力前，到「**模型管理**」把对应模型下好：

- 每个模型显示状态：**可用 / 缺失 / 需额外依赖**。
- 缺失的可**单个下载**，或「**下载全部缺失项**」批量拉取。
- 下载过程有进度条，可取消；失败可重试。

> 💡 字幕擦除、视觉模型等较大的权重会在首次用到时自动下载并校验。介意联网的话，可预先放好权重并用「仅本地模型」相关开关禁用下载（见下文环境变量）。

## 常用环境变量

大多数配置在界面里就能改。少数偏底层的可用环境变量覆盖：

| 变量 | 作用 |
| --- | --- |
| `TRANSLIP_CACHE_DIR` | 模型缓存与流水线产物的根目录（默认 `~/.cache/translip`）。 |
| `DEEPSEEK_API_KEY` | DeepSeek 翻译 / 助手 / 评测所需。也可在界面里填。 |
| `MOSS_TTS_NANO_CLI` | 默认 TTS 后端的外部 CLI 路径。 |
| `SUBTITLE_ERASE_LOCAL_MODELS_ONLY` | 设为 1 禁止字幕擦除权重联网下载（需预放）。 |
| `VISION_BACKEND` | 视频画面感知后端（`auto` / `mlx` / `ollama`）。 |

> 📖 完整的环境变量与默认值清单见仓库 `README.md` 与 `CLAUDE.md`。

## 高级配置

「高级配置」标签下是各阶段（ASR、TTS、翻译、配音修复、时长贴合、混音、输出、OCR、字幕擦除…）的默认参数。改动后底部会出现未保存提示，可「保存配置」或「撤销」。

> 💡 这里改的是**默认值**——新建任务时仍可在向导里针对单条任务覆盖。没有特殊需求，保持默认即可。
