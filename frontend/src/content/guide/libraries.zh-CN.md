# 作品库与角色库

配同一部剧的多集时，你不会想每集都重新认一遍人物和音色。**作品库**和**角色库**就是把这些沉淀下来、跨任务复用的地方。

- **作品库**：管理「作品」——一部剧 / 一部电影这样的顶层容器，可关联多集与多个角色。
- **角色库**：管理「角色」——把人物身份（名字、演员、性别、头像…）和音色 / 说话人对应起来，形成一份**角色 → 说话人**的台账，配音时复用。

## 作品库

入口：左侧「**作品库**」。

![作品库：作品 / 剧集资产](/guide/works.png)

*图 1：作品库以卡片展示每部作品——封面、类型、年份、关联角色数。*

每张卡片是一部作品，可搜索、按类型筛选。你可以：

| 操作 | 说明 |
| --- | --- |
| **新建作品** | 手动录入，或从 **TMDb** 导入（需配好 TMDb 密钥）。 |
| **编辑** | 改标题、类型、年份、别名、封面、标签、备注。 |
| **管理角色** | 跳到角色库并自动过滤到这部作品的角色。 |
| **删除** | 删除作品（会提示关联角色数）。 |

> 💡 从 TMDb 导入时可一并拉取演职表，快速把主要角色建进角色库——省去手动一个个录。TMDb 密钥在[全局设置](/guide/settings)里配。

## 角色库

入口：左侧「**角色库**」。

![角色库：角色 → 说话人台账](/guide/character-library.png)

*图 2：角色库以表格列出角色——头像、名字、演员、性别与操作。*

每一行是一个角色。可按作品过滤（全部 / 未指定 / 某部作品），可搜索名字 / 演员 / 别名。新建或编辑角色时可填：

| 字段 | 说明 |
| --- | --- |
| **名字 / 演员 / 角色** | 人物身份信息。 |
| **性别 / 年龄段** | 影响音色匹配与合成。 |
| **头像 / 颜色** | 用 emoji、首字或远程图片做头像，配色便于区分。 |
| **所属作品** | 绑定到某部作品，或留「不指定」做跨剧复用。 |
| **别名 / 标签 / 备注** | 检索与备注用。 |

## 任务怎么用上它们

- 新建任务时勾「**自动绑定作品**」，任务完成后会自动关联到对应作品。
- 配音流程会参考角色库里的「角色 → 音色」对应，让同一人物在不同集 / 不同任务里**保持一致的音色**。
- 在[配音编辑台](/guide/dubbing-editor)换音色时，也能从沉淀好的角色资产里挑。

> 💡 作品库与角色库是**可选**的——单条任务不建它们也能跑。它们的价值在「同一部剧多次配音」时才真正显现。
