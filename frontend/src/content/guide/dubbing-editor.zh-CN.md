# 配音编辑台

「配音编辑台」（配音工坊）是一个可视化的逐段校对台。它把整条配音里**需要你看的段落**排成一个问题队列，你顺着队列逐段处理：试听、改译文、调时长、换音色、单段重新合成，所有改动一键回写到任务草稿。

进入方式：在**任务详情页**点「进入配音编辑台」，或任务列表里对已完成任务选择进入。

![配音编辑台全貌：问题队列、时间线、检视面板](/guide/dubbing-editor-full.png)

*图 1：三栏布局——左侧问题队列，中间视频与时间线，右侧检视面板。*

## 三栏布局

| 区域 | 作用 |
| --- | --- |
| **问题队列**（左） | 按严重度排好的待处理片段；P0 / P1 / P2 表示优先级。点一条即定位到该片段。 |
| **视频 + 时间线**（中） | 上方是带字幕的视频播放器，下方是原声 / 背景音 / 配音片段的波形轨道。 |
| **检视面板**（右） | 当前片段的全部细节：原文、译文、时长贴合、音色、质量分、风险标记与操作按钮。 |

顶部还有几组切换：**审稿 / 视频优先 / 时间线 / 角色音色 / 预览**视图标签，以及**编辑 / 预览**模式开关；右上角是「导出」与撤销 / 重做。

## 一段的处理流程

选中队列里的一条问题后，在检视面板里：

1. **试听**——「选段试听」播放这一段的配音；可调倍速对比。
2. **看问题**——面板会标出风险：音色不匹配、说话人归因存疑、节奏过快 / 过慢、漏字等。
3. **改文本**——直接编辑译文（可切原文 / 译文 / 双语显示）。改完会**实时预测新时长**能否贴合原片窗口。
4. **调整 / 重合成**——必要时单段「**重新合成**」（可指定语速），或在候选里**换一个音色**。
5. **定状态**——确认无误点「**批准**」，拿不准点「**标记需复核**」。

> 💡 改动是**非破坏性**的：它们写进任务草稿，可随时撤销 / 重做，也能在不满意时整段重来。完成后把草稿交付即可。

## 键盘快捷键

熟练后基本可以脱离鼠标。点右上角键盘图标可随时调出这张表：

![键盘快捷键浮层](/guide/dubbing-shortcuts.png)

*图 2：编辑台的键盘快捷键。*

| 按键 | 动作 |
| --- | --- |
| `↓` / `J` | 下一条问题 |
| `↑` / `K` | 上一条问题 |
| `Space` | 播放 / 暂停配音片段 |
| `A` | 批准当前片段 |
| `F` | 标记需复核 |
| `R` | 局部渲染（重合成当前段） |
| `Ctrl+Z` / `Ctrl+Y` | 撤销 / 重做 |
| `Esc` | 取消选择 |

## 换音色

当某个说话人的音色不理想时，检视面板会给出**候选音色列表**。试听对比后选定一个，编辑台会用新音色重合成相关片段。若某段实在不适合克隆，可标记为不可克隆，让它走兜底处理。

## 导出字幕

校对过程中可随时用「导出 SRT」把当前文本导出成字幕文件，便于在别处复用或质检。

> 📖 想要更细的逐功能说明？编辑台内置一份操作手册，点界面里的「使用手册 / 帮助文档」即可打开（也可直接访问 `/manual.html`）。
