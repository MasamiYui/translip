# 常见问题与排查

跑不动、报错、结果不对？先在这里找一遍。

## 故障排查

| 现象 | 原因 / 解法 |
| --- | --- |
| **找不到 FFmpeg / 分离·混音·合并报错** | FFmpeg 没装或不在 `PATH` 上。安装后确保命令行能直接调用 `ffmpeg`。 |
| **语音合成失败、提示缺少 moss-tts-nano** | 默认 TTS 后端依赖外部 `moss-tts-nano` CLI，需单独安装；或在新建任务 / 工具里换一个 TTS 后端。 |
| **字幕识别（OCR）报 ImportError** | OCR 是可选依赖。用 `uv sync --extra ocr` 安装后再跑。 |
| **翻译 / 助手 / 翻译打分报缺少 Key** | 这些用到 DeepSeek，需在[全局设置 → API 密钥](/guide/settings)填 `DEEPSEEK_API_KEY`。 |
| **门控模型下载 401 / 无权限** | 在设置里填好 HuggingFace 令牌并**重启服务**。 |
| **模型下载失败 / 中断** | 检查网络后在「模型管理」里重试；介意联网可预放权重并开启「仅本地模型」。 |
| **某阶段失败但不知为何** | 在任务详情页点该阶段节点，展开**日志**与**错误信息**。 |
| **改了设置但重跑像没生效** | 只有改动会影响缓存键的设置才会触发重算。想强制重算就改相关参数，或新建任务时关掉「使用缓存」。 |

## 常见疑问

**必须联网吗？**
不必。translip 本地优先，模型默认在本地跑。只有这几件事需要联网：DeepSeek（翻译 / 助手 / 评测）、TMDb（作品导入），以及首次下载模型权重。

**界面能切英文吗？**
能。右上角切换**中文 / English**。这份文档也会跟着切换章节框架（正文以中文为主）。

**配音音色不太像原声怎么办？**
默认 TTS 后端是轻量本地模型，音色还原有上限。可以：在[配音编辑台](/guide/dubbing-editor)给该说话人**换音色**、新建任务时**换一个 TTS 后端**、或在[角色库](/guide/libraries)里沉淀更合适的音色参考。

**任务状态「部分成功」是什么意思？**
主链路出了成片，但某些**可选支线**（如字幕擦除）没完全跑通。通常仍可交付，必要时单独重跑那条支线。

**我只想做某一步（只分离 / 只识别字幕），不想跑整条流水线？**
用[原子工具](/guide/atomic-tools)；或者用 [AI 助手](/guide/ai-assistant)一句话说清需求。

**产物和缓存放在哪？**
默认在 `~/.cache/translip` 下（可用 `TRANSLIP_CACHE_DIR` 改）。流水线产物在 `output-pipeline/<task_id>/` 里，按 `分离 / 转写 / … / 交付` 分目录。

**清理缓存安全吗？**
「系统信息」里的清理缓存会删掉缓存产物——已交付到别处的成片不受影响，但任务的中间产物会没，下次相关阶段需重算。删除某个任务则会清掉它自己的产物目录。
